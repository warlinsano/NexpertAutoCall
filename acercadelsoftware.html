¿Que es NexpertAutoCall?

NexpertAutoCall es una aplicación escrita en PHP para que Asterisk envíe transmisiones de voz. Puede generar múltiples llamadas simultáneas a una lista de números y reproducir un mensaje de audio personalizado. NexpertAutoCall puede recibir opciones de entrada DTMF y conectarse a cualquier destino predefinido. Podemos encontrar diferentes ocasiones en las que Callblaster puede resolver problemas con el mínimo esfuerzo, como la gestión de eventos, la relación con el cliente, la gestión del personal y donde la gente quiera saber de usted. NexpertAutoCall es simple en diseño e interfaz de usuario para que cualquiera pueda instalar y comenzar a realizar llamadas en pocos minutos. Los siguientes párrafos lo convertirán en un experto de esta aplicacion de llamadas.

NexpertAutoCall es una aplicación desarrollada por el Ing. Warlin Sano y el Ing. Jonathan Salvador como una actulizacion o tomando como base el sistema Callblaster que se desarrollo en Nethram. Nethram es un innovador de telecomunicaciones y nube con sede en Silicon Valley. Cuya misión es proporcionar las soluciones de telefonía en la nube que utilizan los grandes a un precio asequible, escuchando sus necesidades y proporcionándole lo que realmente desea. Gracias por usar NexpertAutoCall, no dude en escribirnos a warlinsano@gmail.com para soporte de esta plataforma en linea.

¿Características Princiaples?


  • Soporta múltiples llamadas simultáneas a una lista de números.
  • Posibilidad de mensajes de audio separados para cada número.
  • Recibe entradas DTMF.
  • Destinos de redireccionamiento configurables a cualquier contexto y extensión de asterisco (número externo o interno, grupos de timbre, cola, ivr, etc.).
  • La pantalla de marcado muestra el estado de la llamada y la respuesta DTMF.
  • Registro de llamadas detallado con deposición de llamadas y DTMF recibido.
  • Utiliza el enrutamiento de salida regular para la transmisión, use cualquier troncal o patrón de marcado.
  • Fácil de usar el panel de administración.
  • Entre otras caracteristicas.

¿Requisitos Previos?


  • Servidor del asterisk
  • Con tronco y ruta de salida para realizar llamadas salientes.
  • Freepbx, "opcional"
  • Base de datos MySQL
  • Servidor web
